Windows Defender



Microsoft Defender Antivirus (cunoscut sub numele de Windows Defender Antivirus înainte de Windows 10 mai 2020 Update sau Windows Defender înainte de Windows 10 Creators Update) este o componentă anti-malware a Microsoft Windows. A fost lansat pentru prima dată ca un program anti-spyware gratuit descărcabil pentru Windows XP și a fost livrat ulterior cu Windows Vista și Windows 7. A evoluat într-un program antivirus complet, înlocuind Microsoft Security Essentials, ca parte a Windows 8 și versiunile ulterioare.

Înainte de Windows 8, Windows Defender proteja utilizatorii numai împotriva programelor spyware. Acesta include o serie de agenți de securitate în timp real care monitorizează mai multe zone comune ale Windows pentru modificări care ar fi putut fi cauzate de spyware. De asemenea, are capacitatea de a elimina software-ul ActiveX instalat. Windows Defender a oferit un suport integrat pentru Microsoft SpyNet care permite utilizatorilor să raporteze către Microsoft ceea ce consideră că sunt spyware și ce aplicații și drivere de dispozitiv permit instalarea pe sistemele lor. Protecția împotriva virușilor a fost adăugată ulterior în Windows 8; care se aseamănă cu Microsoft Security Essentials (MSE). De asemenea, folosește același motor anti-malware și definiții de virus din MSE.

În Windows 10, setările Windows Defender sunt controlate în Centrul de securitate Windows Defender. În Windows 10 Anniversary Update, este introdusă o nouă siglă și va apărea o notificare pop-up pentru a anunța rezultatele unei scanări, chiar dacă nu sunt găsiți viruși.
